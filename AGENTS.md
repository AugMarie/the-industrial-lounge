# AGENTS.md

## Project
The Industrial Lounge website.

## Purpose
This is a lightweight static website for The Industrial Lounge, a classic industrial music, EBM, darkwave, and new wave online station.

## Core Rules
- Preserve The Industrial Lounge branding.
- Preserve the TIL logo.
- Preserve Mixcloud links and embeds.
- Keep the website lightweight.
- Prefer static HTML, CSS, and JavaScript.
- Do not add React, Vue, Angular, Next.js, or other frameworks.
- Do not remove existing content without approval.
- Do not commit without approval.
- Do not push without approval.

## Design Direction
The site should feel like an underground radio station, not a corporate website.

## Deployment Warning
Cloudflare Pages automatically deploys when changes are pushed to main.
Treat pushes as production deployments.
