# The Industrial Lounge - Project Status

## Project links

- **Project name:** The Industrial Lounge
- **GitHub repository:** https://github.com/AugMarie/the-industrial-lounge
- **Website:** https://www.theindustriallounge.com
- **Mixcloud:** https://www.mixcloud.com/DJAapocalypse/
- **Live stream:** https://www.mixcloud.com/live/DJAapocalypse/

## Purpose

The Industrial Lounge website is the front door to a 24/7 online radio station and streaming platform focused on classic industrial music, EBM, darkwave, and new wave. Its primary purpose is to direct listeners to the live stream and archived mixes.

## Current status

The lightweight static website has been imported into GitHub and is synchronized on the `main` branch. It currently includes a responsive homepage, direct access to the Mixcloud live stream and archived mixes, and a support page with a forthcoming merchandise placeholder.
